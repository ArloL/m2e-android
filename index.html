Getting Started

The Android Configurator is an Maven Integration for Eclipse (m2e) plug-in that adds maven support for Android Developer Tools (ADT) and the Maven Android Plugin. It brings the power of Maven dependency management into the ADT.

1. Before you start

Before you start you will need Eclipse Helios or Indigo
installed, as well as some additional Eclipse plug-ins. Instructions for installing these components can be found here:

Android SDK and ADT 12.0.0+
Maven Integration for Eclipse 1.0.0+

Note: If you have an old, alpha 0.2.x version of m2elipse-android-integration installed you should uninstall it before installing new 0.3.x beta versions.

2. Create Eclipse project

If you already have an Android project please make sure you have created a POM for your project using version 3.0.0-alpha-2 or greater of the maven-android-plugin. Right-click on your project and select Configuration -> Convert to Maven Project.

If you are starting with a new project you can use the Maven Android archetypes to create Android projects completely within Eclipse:

  1. Create a new Maven Project (File -> New -> Project... then select Maven -> Maven Project).
  2. When prompted to Select Archetype click Add Archetype...
  3. In the dialog that appears enter "de.akquinet.android.archetypes" for Archetype Group Id.
  4. In Archetype Artifact Id enter "android-quickstart"
  5. In Archetype Version enter "1.0.6" and contiune
  6. When prompted enter your desired project group and artefact ID, version and, optionally, set the "platform" property for the Android version (defaults to '7')
  7. Click Finish
  8. Open newly created POM in editor and change the maven-android-plugin version to 3.0.0-beta-2
  9. Right-click on new project and select Maven -> Update Project Configuration

3. Install Android Connector

Your newly created project should have errors complaining that "Plugin execution not covered by lifecycle configuration". To fix this and install the Android Connector simply open your project POM and click on the error. This should give you the option to Discover new m2e connectors. Select this and you should be automatically prompted to download and install the Android Connector.

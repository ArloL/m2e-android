Getting Started

The Android Configurator is an Maven Integration for Eclipse (m2e) plug-in that adds maven support for Android Developer Tools (ADT) and the Maven Android Plugin. It brings the power of Maven dependency management into the ADT.

1. Before you start

Before you start you will need Eclipse Helios or Indigo installed, as well as some additional Eclipse plug-ins. Instructions for installing these coponents can be found here:

Eclipse 3.6+
Android Developer Tools 12.0.0+
Maven Integration for Eclipse 1.0.0+
Maven Android Plugin 3.0.0+
